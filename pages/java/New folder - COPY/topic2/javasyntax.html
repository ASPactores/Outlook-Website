The main() Method

The main() method is required and you will see it in every Java program:

Code inside the main will be executed. Meaning, code inside the main will happen, no matter what (except for errors, of course).

For the last part of your example, the print(ln) method was used, which can print out lines of text you have stated in your code. In this example, it is "Hello World."

Remember

Always remember the following:

  • The curly braces {} marks the beginning and the end of a block of code.
  • Each code statement must end with a semicolon.
  • You should run your code in blocks following classes
  • Code in your main() method will always run, unless it has error

Note: Java is a strict programming language. Forget one semicolon and you're bound to error.
