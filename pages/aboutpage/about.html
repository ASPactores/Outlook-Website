ABOUT

WHAT IS OUTLOOK!?

OUTLOOK! is a project created by Anakin Pactores and Marc Balabat with the primary goal of teaching programming languages to everyone for free.

The website contains lessons that are designed to be simple and easy to follow so that anyone regardless of age and skill-level can start learning how to program.

SELF-PACED LEARNING

The learning resources are available to you 24/7. You can learn at your own pace. Creating an account will allow you to track your learning progress.

INSTANT APPLICATION OF KNOWLEDGE

The website offers practice problems for each lesson and course. These problems allow the learners to verify what they learned and apply the skills they acquired from the learning resources.
